# Streamer.bot Twitch Extensions

This repo contains various extensions for Streamer.bot that can integrated into your Twitch streams. Everything is 100% and neatly displayed on the website https://tawmae.xyz/.

That includes...

- OBS Widgets
- Chat Games
- Integrations like Spotify songrequests or Bluesky posts
- Utility like a currency system, raffles, watchtime
- and much much more!

Feel free to check it out! :)
