FAQ

How do I update extensions?
You update by just re-importing all the actions. All your settings from the Settings UI will persist. You only have to re-enable the commands.
Can I change address or port for overlays?
Yes, every overlay allows to add the parameters ?address, ?port and ?password.
Can I use overlays on a different PC?
Yes, but you either have to use the HTML locally by downloading it or you bypass security settings for browser sources in OBS.
