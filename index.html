FAQ

How do I install an extension?
Each extension page includes an import string for Streamer.bot. Copy and paste it into your Actions tab.
Can I customize overlays?
Yes. Most widgets include editable CSS and variable settings to match your stream layout.
Are these extensions free?
Most are free to use. Some advanced ones may require a commission for custom versions.
Where can I report bugs?
You can reach out via Discord or GitHub under the respective repository issues section.
