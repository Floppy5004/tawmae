HOW TO EXPORT FIREBOT POINTS

1. In Firebot, go to your Settings, then on Backups and create a backup with Backup Now.

2. Click on Manage Backups and then Open Backups Folder.

3. Extract the latest backup .zip file.

4. After extracing, go to profiles -> Main -> db The file we are interested in in users.db. Either copy the filepath or paste the file somewhere you can easily select it later on. But then you're done and you can continue in Streamer.bot.
