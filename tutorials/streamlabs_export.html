--- layout: default title: "STREAMLABS EXPORT | tawmae" description: "A tutorial how to export points from Streamlabs" ---
This only applies if you are using the Streamlabs Desktop Chat Bot. If you use the Streamlabs Twitch Extension for loyalty points, you can skip this. You cannot export points from the Streamlabs Cloud Bot.

1. Click on the little User Icon in the bottom left corner.

2. Go to Streamlabs and then make sure you are connected. If you generated a new token, make sure to not copy it via the Copy icon on their website, because it doesn't work and will result in an invalid token. Make it visible and copy it manually instead. God damn, Streamlabs sucks.

3. Finally, click EXPORT CURRENCY & HOURS TO EXTENSION, confirm all dialogs and let it export. Then you're done and can continue in Streamer.bot.
