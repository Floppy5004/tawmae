MOD TOOLS

Site under construction, this is just to see the mod commands.

COMMANDS

URL Filter

!permit [User]

Permit a specified user to post links to chat for a given amount of time.

!whitelistURL [URL]

Add a link to the whitelist.

!removeWhitelistURL [URL]

Removes a link from the whitelist.

Follow Bot Attacks

!followShield [on/off]

Toggles the Follow Bot Attack shield mode.

!followAttackUndoAll

Unblocks and unbans every user that got caught during an active Follow Attack Shield. This is just a safety measure.

!removeAllFollowers [Minutes]

Removes all followers that have followed within the last X minutes. Also works if Streamer.bot was not active during the follows.

Hate Raid

!hateShield [on/off] or !hateRaid [on/off]

Toggles the hate raid shield.

Channel Management

!addCommand [!command] [Command Text]

Adds a custom command with a simple text output. Can contain %user% and %userName% variables.

!editCommand [!command] [Command Text]

Edits a custom command.

!removeCommand [!command]

Removes a custom command.

!customCommands

Shows all custom commands.

!addCounter [!command] [Command Text]

Adds a custom counter with a simple text output. Can contain %count%, %user%, and %userName% variables. Use “+” to increment or “–” to decrement. E.g., !command +1 or !command -1. You can also set it directly: !command 50.

!editCounter [!command] [Command Text]

Edits a custom counter.

!removeCounter [!command]

Removes a custom counter.

!customCounters

Shows all custom counters and their current values.

!channelTitle [Title]

Set a channel title.

!channelGame [Game] or !channelCategory [Category]

Set the channel game/category.

!channelSetTags [Tag1] | [Tag2] | [Tag3] ...

Set up to 10 channel tags, separated by a | symbol.

!channelAddTag [Tag]

Add a channel tag.

!channelRemoveTag [Tag]

Remove a channel tag.

!channelClearTags

Clear all channel tags (Note: Twitch API requires at least one tag, so a “No Tags” placeholder will be added).

OBS Management

!scenes

Lists all OBS scene names in chat.

!setScene [Scene Name] or !changeScene [Scene Name]

Set a specific active OBS scene.

!endStream

Ends the stream (double-confirmation required if enabled).

Raids and Shoutouts

!raid [User]

Raid a channel.

!raidCancel or !cancelRaid

Cancel an ongoing raid.

!raidMessage

Posts a custom raid message into chat (for users to copy when you’re about to start a raid).

!modShoutout [User]

Posts a shoutout into chat and also does the Twitch integrated shoutout.

Chat Management

!modTranslate [Text]

Translates a given text into a chosen language (also works with just !modTranslate when used in a reply).

!clearChat

Clear chat.

!warn [User] [Reason]

Warn a specified user. The reason is optional.

!timeout [User] [Duration] [Reason]

Timeout a specified user. Duration (seconds) and reason are optional.

!ban [User] [Reason]

Ban a specified user. The reason is optional.

!unban [User]

Unban a specified user.

!purge [User]

Deletes all messages of a specified user.

!addBlockedTerm [Term]

Adds a blocked term to the Twitch internal block list.

!removeBlockedTerm [Term]

Removes a blocked term from the Twitch internal block list.

!modRunAd [Duration]

Runs a commercial for the specified duration (max. 180 seconds, defaults to 180 if exceeded).

!slowMode [Delay (seconds)]

Toggles chat slow mode with the specified delay in seconds.

!followerMode [Delay] or !followersOnly [Duration]

Toggles followers-only chat with the specified minimum follow duration. If [Duration] is a number, it defaults to hours. You can specify “min”, “h”, or “d”, e.g., !followerMode 1d.

!emoteOnly [on/off]

Toggles emote-only chat.

!subMode or !subOnly

Toggles subscribers-only chat.

!ttsStop or !stopTts

Stops the current TTS queue.

!ttsResume or !resumeTts

Resumes the TTS queue.

!ttsClear or !clearTts

Clears the TTS queue.

!ttsPause or !pauseTts

Pauses the TTS queue.

!autoMod [Level]

Enables AutoMod from level 0 to 4.

Channel Point Rewards

!rewardEnable [Reward Name]

Enables a specified channel point reward.

!rewardDisable [Reward Name]

Disables a specified channel point reward.

!rewardPause [Reward Name]

Pauses a specified channel point reward.

!rewardUnpause [Reward Name]

Unpauses a specified channel point reward.

!rewardTitle [Reward Name] | [Title]

Sets the title of a specified channel point reward, separated by a | symbol.

!rewardPrompt [Reward Name] | [Prompt] or !rewardDescription [Reward Name] | [Cost]

Sets the prompt/description of a specified channel point reward, separated by a | symbol.

!rewardCost [Reward Name] | [Cost]

Sets the cost of a specified channel point reward, separated by a | symbol.

Predictions

!predictionCreate [Title] | [Duration] | [Outcome1] | [Outcome2] ...

Creates a channel point prediction with up to 10 outcomes. Requires a title and at least two outcomes. Duration is optional. Separated by a | symbol.

!predictionLock

Locks the currently active prediction.

!predictionCancel

Cancels the currently active prediction.

!predictionResolve [Outcome Number] or [Outcome Title]

Resolves a prediction after it’s been locked. Can use an outcome number or the actual outcome title.

Polls

!pollCreate [Title] | [Duration] | [Option1] | [Option2] ...

Creates a poll with up to 5 options. Requires a title and at least two options. Duration is optional. Separated by a | symbol.

!pollEnd

Ends the current poll.

!pollArchive

Archives the current poll.
