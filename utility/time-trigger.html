Run actions at specific times and/or dates.

SETUP

1. Download the .dll file for the settings UI and place it into your /dlls folder of your Streamer.bot directory. If you already have it from another extension of mine, you can skip this step.

Download and detailed instructions here.

2. Copy the Action Import text from above into your clipboard.

3. In Streamer.bot, click the Import button at the top of the window. Paste the text into the Import String section and press Import at the bottom.

4. In the action [TIME TRIGGER] 1 - Settings, rightclick the Test trigger and hit Test Trigger to open the Settings UI.

You can add your times and optionally dates in there and choose an action that should run when the time comes. The date conditions are optional, but you can choose to run an action every day of the time (then just leave all unchecked) or you can run the action at this day on a specific date, day of the week or xth day of the month.

Restart Streamer.bot afterwards (you only need to do this after the first import, not every time)

Then you're done! Pog Clap
