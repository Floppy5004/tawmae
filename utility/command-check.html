COMMAND CHECK

Your viewers aren't able to correctly spell your commands? This extension not only allows to neatly display all your available commands to chat, but it also detects commands in your chat that don't exist and provides a list of commands instead.

• Check for non-existing commands in chat
• Display all available commands
• Easily choose which commands to display

SETUP

1. Copy the Action Import text from above into your clipboard.

2. In Streamer.bot, click the Import button at the top of the window. Paste the text into the Import String section and press Import at the bottom.

3. Now head to the Commands tab and look for the imported commands. Then rightclick the group and select Group -> Enable All.

4. In the action [COMMAND CHECK] 0 - Settings rightclick the Test trigger and hit Test Trigger to open the Settings UI. In the General tab, choose the command groups you want to include when displaying all commands. You can select < ALL GROUPS >. Save afterwards.

And then you're done!
